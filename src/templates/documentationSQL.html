Présentation du MLD

Le Modèle Logique de Données (MLD) présenté ci-dessous représente la structure d'une base de données pour gérer les réservations d'hôtel. Voici une description des principales tables et de leurs relations :

  • CLIENT : Contient les informations sur les clients, comme leur nom, prénom, date de naissance, et coordonnées.
  • RESERVATION : Stocke les détails des réservations, tels que la date de début, le nombre de nuits, et le mode de paiement.
  • CHAMBRE : Représente les chambres disponibles dans l'hôtel avec des détails comme l'étage, la superficie, la capacité, et les équipements.
  • OCCUPATION : Indique quelle chambre est occupée pour une réservation donnée.
  • PERIODE : Définit les périodes spécifiques (exemple : haute saison).
  • PRIX : Associe un tarif à une chambre en fonction d'une période.

SELECT

La commande SELECT permet de récupérer des données depuis une table dans une base de données. Elle est utilisée pour afficher des colonnes spécifiques ou toutes les colonnes d'une table.

Exemple : Afficher les noms, prénoms et pays de tous les clients :

SELECT nom, prenom, pays
FROM CLIENT;

Exemple de résultat :

nom prenom pays
Dupont Jean France
Martin Marie Canada
Rossi Luca Italie
Smith John États-Unis

FROM

La clause FROM spécifie la table où chercher les données. Elle est obligatoire dans une requête SQL.

Exemple : Afficher toutes les informations des réservations :

SELECT *
FROM RESERVATION;

Exemple de résultat :

numero dateDebut nbNuits datePaiement modePaiement client
1001 2024-11-01 3 2024-10-20 Carte 1
1002 2024-11-05 5 2024-10-22 PayPal 2
1003 2024-11-10 2 2024-10-25 Virement 3

WHERE

La clause WHERE permet de filtrer les résultats selon une ou plusieurs conditions.

Exemple : Trouver les clients qui habitent en France :

SELECT nom, prenom
FROM CLIENT
WHERE pays = 'France';

Exemple avec plusieurs conditions : Trouver les chambres avec une capacité de 2 personnes et une baignoire :

SELECT numero, etage
FROM CHAMBRE
WHERE capacite = 2 AND baignoire = 'oui';

Exemple de résultat :

nom prenom
Dupont Jean
